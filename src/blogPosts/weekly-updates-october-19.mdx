### 🔬 5 Tips to Make Your Lambda Functions Run Faster (and Cheaper)

The AWS Lambda service allows us to easily deploy and run our own code, without worrying too much about the underlying infrastructure (when compared to non-serverless technologies).

But, there are still a couple of useful tricks to know, when it comes to making your code execute as fast as possible,
which is especially important when creating synchronous workflows like for example HTTP APIs.

In this short <ExternalLink href="https://www.webiny.com/blog/5-tips-to-make-your-lambda-functions-run-faster-and-cheaper?utm_source=Webiny-blog&utm_medium=webiny-blog&utm_campaign=webiny-weekly-updates-blog-oct-19&utm_content=webiny-5-tips-lambda-functions-run-faster-cheaper&utm_term=W00231"> article</ExternalLink>, Webiny’s Lead Developer <ExternalLink href="https://twitter.com/doitadrian">Adrian</ExternalLink> will show you
five tips to make your Lambda functions perform not only faster, but also cheaper! 🚄
